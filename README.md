# LogiX 

OOP project program

Примерна архитектура: 

| Слой       | Отговаря за                           |
| ---------- | ------------------------------------- |
| Reader     | Чете редове (синхронно или live)      |
| Parser     | Превръща реда в `LogEntry`            |
| Aggregator | Обединява по време, филтрира, сортира |
| Output     | Извежда резултата                     |
